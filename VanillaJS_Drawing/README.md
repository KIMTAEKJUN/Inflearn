> ## **강의 들으면서 몰랐던거 구글링해서 메모하기**

<br/>

+ strokeStyle
  + 도형의 윤곽선 색을 설정
``` JS
ctx.strokeStyle = "color";
```
<br/>

+ stroke
 + 윤곽선을 이용하여 도형을 그림
``` JS
ctx.stroke();
```

<br/>

+ lineWidth
  + 현재 선의 두께를 설정, 초기 설정값은 1.0 단위
``` JS
ctx.lineWidth = value;
```

<br/>

+ beginPath
  + 도형을 그리기 시작
``` JS
ctx.beginPath();
```

<br/>

+ moveTo
  + 선이 시작될 좌표를 설정
``` JS
ctx.moveTo(x, y);
```

<br/>

+ lineTo
  + 선이 끝나는 좌표를 설정
``` JS
ctx.lineTo(x, y);
```
