> ## **강의 들으면서 몰랐던거 구글링해서 메모하기**

> #### .padStart(2, "0") → 주어진 길이를 만족하는 새로운 문자열을 채움 (왼쪽부터 채움)
> #### localStorage.setItem("key", value) → 키에 데이터 쓰기
> #### localStorage.getItem("key) → 키로 부터 데이터 읽기
> #### JSON.stringify → 값이나 객체를 JSON 문자열로 변환
